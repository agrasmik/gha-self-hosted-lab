## Общие параметры для всего Workflow

В каждом Workflow должен минимально необходимый набор объявлений: 
- on
- jobs

Остальные (*name* и *run-name*) - не обязательные параметры. 

### name:
**name** - это название Workflow. Оно будет отображаться в списке всех доступных Workflows проекта в разделе Actions в репозитории GitHub. 
Как водится, можно задать любое, но осмысленное.

### run-name:
Отображаемое имя. Это имя можно увидеть в списке запущенных Workflow. Нужно обратить внимание на то, что уже здесь можно использовать встроенные переменные GitHub Actions, создаваемые в момент запуска Workflow.

### on:
Список Событий, на которые будет реагировать данный WorkFlow. 

### Jobs:
Список заданий, которые необходимо выполнить в рамках текущего WorkFlows.
